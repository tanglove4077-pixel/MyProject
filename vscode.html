💻 Visual Studio Code

โปรแกรม Code Editor ที่ทรงพลังสำหรับนักพัฒนา

📖 VS Code คืออะไร?

Visual Studio Code (VS Code) เป็นโปรแกรมแก้ไขข้อความ (Source Code Editor) ที่ได้รับความนิยมสูงสุด ในโลกของการพัฒนาซอฟต์แวร์ ออกแบบโดย Microsoft โดยเป็นโปรแกรมฟรีและโอเพนซอร์ส (Open Source)

VS Code มีความเบา เร็ว และสามารถขยายได้ด้วย Extensions ทำให้เหมาะสำหรับนักพัฒนา ในส่วนมากที่ต้องการ Code Editor ที่ปลอดภัยและมีความยืดหยุ่นสูง

📥 ดาวน์โหลด VS Code

🚀 วิธีการใช้งาน VS Code

1. สร้างโฟลเดอร์โปรเจค

mkdir my-project
cd my-project

2. เปิด VS Code ในโฟลเดอร์นั้น

code .

3. สร้างไฟล์ใหม่

  • คลิกปุ่ม "New File" ในเมนู File หรือกด Ctrl + N
  • ตั้งชื่อไฟล์ และบันทึก

4. เริ่มเขียนโค้ด

  • พิมพ์โค้ด HTML, CSS, JavaScript
  • ใช้ IntelliSense เพื่อความสะดวก
  • บันทึกไฟล์เมื่อเสร็จ

🧩 Extensions ที่แนะนำ

สำหรับการพัฒนา Web:

1. Live Server

ช่วยให้คุณสามารถ Preview ไฟล์ HTML ที่ทำการเปลี่ยนแปลงแบบ Real-time ได้

2. Prettier - Code formatter

ช่วยให้โค้ดของคุณดูสวยงาม และมีรูปแบบที่สม่ำเสมอโดยอัตโนมัติ

3. ES7+ React/Redux/React-Native Snippets

ให้ Snippets สำหรับการพัฒนา React Applications

4. Thunder Client

ใช้สำหรับ Test API และ HTTP Requests

5. GitLens

ช่วยให้คุณเห็นข้อมูล Git และ Commit History ได้อย่างชัดเจน

6. Path Intellisense

ให้ Auto-complete สำหรับ File Paths

วิธีติดตั้ง Extensions:

  • คลิกไอคอน Extensions ที่แถบด้านซ้าย หรือกด Ctrl + Shift + X
  • พิมพ์ชื่อ Extension ที่ต้องการ
  • คลิกปุ่ม "Install"

🎨 Themes และการปรับแต่ง

วิธีเปลี่ยน Theme:

  • ไปที่ File → Preferences → Color Theme
  • หรือกด Ctrl + K Ctrl + T
  • เลือก Theme ที่ชอบ

Themes ที่นิยม:

  • Dracula - ธีมสีม่วงเข้มและเรียบร้อย
  • One Dark Pro - ธีมสีเข้มและใช้งานง่าย
  • Material Theme - ธีมสมัยใหม่จาก Material Design
  • Nord - ธีมสีเย็นสุดสวย
  • Atom One Dark - คล้ายกับ Editor Atom

💡 คำแนะนำในการใช้งาน

  • ใช้ Code Formatting: บันทึกไฟล์จะมี Format โค้ดโดยอัตโนมัติ
  • ใช้ Git Integration: Version Control ตรงใน VS Code
  • ใช้ Emmet Abbreviation: เขียน HTML ได้เร็วขึ้น เช่น div.container>h1+p
  • ใช้ Multiple Cursors: กด Ctrl + D เพื่อเลือกคำเดียวกันหลายบรรทัด
  • ใช้ Command Palette: กด Ctrl + Shift + P เพื่อเข้าถึงคำสั่งทั้งหมด
  • ใช้ Workspace Settings: ตั้งค่า Settings ต่างๆ ตามความต้องการ
