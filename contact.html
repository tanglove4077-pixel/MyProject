❓ คำถามที่พบบ่อย (FAQ)

ฉันจะเริ่มต้นใช้งาน MyProject ได้อย่างไร?
ไปที่หน้า ติดตั้ง และทำตามขั้นตอนการติดตั้ง ซอฟต์แวร์ที่จำเป็น จากนั้นเข้าสู่แพลตฟอร์มเพื่อเริ่มเรียนรู้
ฉันสามารถใช้งาน MyProject บนอุปกรณ์มือถือของฉันได้หรือไม่?
MyProject ออกแบบให้ Responsive และสามารถใช้งานบนเว็บเบราว์เซอร์ของมือถือได้ แต่ขอแนะนำให้ใช้คอมพิวเตอร์สำหรับประสบการณ์ที่ดีที่สุด
ฉันมีปัญหาในการติดตั้ง ฉันควรทำอย่างไร?
คุณสามารถติดต่อเราผ่านแบบฟอร์มข้างบนหรือส่งอีเมลไปยัง support@myproject.com พร้อมอธิบายปัญหาของคุน
ข้อมูลของฉันจะปลอดภัยหรือไม่?
ใช่ เรามีมาตรการรักษาความปลอดภัยอย่างเข้มงวด ข้อมูลของคุณจะไม่ถูกแชร์ให้บุคคลที่สาม และอยู่ภายใต้นโยบายความเป็นส่วนตัว
ฉันสามารถให้ความเห็นหรือเสนอฟีเจอร์ใหม่ได้หรือไม่?
ได้ เราต้องการความเห็นจากผู้ใช้! กรุณาใช้แบบฟอร์มข้างบนเพื่อส่งความคิดเห็น หรือไปที่ GitHub Repository เพื่อเสนอ Feature Requests

เรหวังว่าได้ยินจากคุns เร็ว ๆ นี้! 🎉
