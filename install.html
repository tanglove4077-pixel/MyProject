💻 Installation Guide

คำแนะนำการติดตั้งโปรแกรมที่จำเป็นทั้งหมด

🎯 ข้อกำหนดขั้นต่ำ

ระบบปฏิบัติการที่รองรับ:

  • Windows 10 ขึ้นไป
  • macOS 10.12 ขึ้นไป
  • Linux (Ubuntu, Fedora, etc.)

ความต้องการระบบ:

  • RAM: 4GB ขึ้นไป
  • ไดรฟ์ว่าง: 2GB ขึ้นไป
  • Internet Connection: จำเป็น

📦 ซอฟต์แวร์ที่ต้องติดตั้ง

1️⃣ Visual Studio Code (Code Editor)

โปรแกรมสำหรับเขียนและแก้ไขโค้ด

ขนาด: ~100 MB

หน้าที่: เขียนโค้ด HTML, CSS, JavaScript

📥 ดาวน์โหลด VS Code

2️⃣ Git (Version Control)

ระบบจัดการเวอร์ชันของไฟล์โปรแกรม

ขนาด: ~50 MB

หน้าที่: Commit, Push, Pull โค้ด

📥 ดาวน์โหลด Git

3️⃣ Node.js & npm (JavaScript Runtime)

สภาพแวดล้อมสำหรับรัน JavaScript และ React

ขนาด: ~150 MB

หน้าที่: npm packages, development server

📥 ดาวน์โหลด Node.js

🚀 ขั้นตอนการติดตั้ง

1ติดตั้ง Visual Studio Code

  1. ไปที่ code.visualstudio.com
  2. คลิกปุ่ม "Download" สำหรับระบบปฏิบัติการของคุณ
  3. รัน Installer และตามขั้นตอนการติดตั้ง
  4. เปิด VS Code และตรวจสอบให้แน่ใจว่าทำงาน

2ติดตั้ง Git

  1. ไปที่ git-scm.com
  2. ดาวน์โหลด Git สำหรับระบบปฏิบัติการของคุณ
  3. รัน Installer (ยอมรับการตั้งค่าเริ่มต้นก็ได้)
  4. เปิด Terminal/Command Prompt และพิมพ์:
    git --version

3ติดตั้ง Node.js

  1. ไปที่ nodejs.org
  2. ดาวน์โหลด LTS Version (Long Term Support)
  3. รัน Installer และตามขั้นตอนการติดตั้ง
  4. เปิด Terminal/Command Prompt และพิมพ์:
    node --version
    npm --version

⚙️ การตั้งค่า Environment

สำหรับผู้ใช้ Windows:

  1. เปิด "Environment Variables" (ค้นหา "env" ใน Start Menu)
  2. ตรวจสอบให้แน่ใจว่ามี Path ของ Git และ Node.js
  3. Restart Terminal เพื่อให้เปลี่ยนแปลงมีผล

สำหรับผู้ใช้ Mac/Linux:

โดยปกติจะติดตั้ง Path อัตโนมัติ ไม่ต้องตั้งค่า

✅ Verification Checklist

VS Code ติดตั้งแล้ว
Git ติดตั้งแล้ว (ตรวจสอบด้วย git --version)
Node.js ติดตั้งแล้ว (ตรวจสอบด้วย node --version)
npm ติดตั้งแล้ว (ตรวจสอบด้วย npm --version)
Terminal/Command Prompt ทำงานปกติ

🎓 ขั้นตอนถัดไป

  1. Clone MyProject Repository จาก GitHub
  2. รัน npm install เพื่อติดตั้ง dependencies
  3. รัน npm run dev เพื่อเริ่ม development server
  4. เปิด http://localhost:5173 ในเบราว์เซอร์
⚠️ สำคัญ: บันทึก Node.js เวอร์ชัน LTS (Long Term Support) เพื่อความเสถียร ห้ามใช้เวอร์ชัน Current เนื่องจากอาจมีปัญหาความเข้ากันได้

คุณพร้อมเริ่มทำการเรียนรู้แล้ว! 🎉
