📤 Upload MyProject to GitHub

คำแนะนำทีละขั้นตอนในการอัพเดตโปรเจคไปยัง GitHub

📋 ข้อกำหนดเบื้องต้น

  • ✅ ติดตั้ง Git แล้ว
  • ✅ มี GitHub Account (ถ้าไม่มี ไปสมัครที่ github.com)
  • ✅ โปรเจค MyProject พร้อมใช้งาน

🚀 ขั้นตอนการอัพโหลด

1สร้าง GitHub Repository

  1. ไปที่ github.com และ Login
  2. คลิก "+" ที่มุมบนขวา → "New repository"
  3. ตั้งค่าดังนี้:
    • Repository name: MyProject
    • Description: VS Code Learning Hub
    • Visibility: เลือก Public
  4. คลิก "Create repository"
💡 Tips: เลือก Public เพื่อให้ GitHub Pages ทำงาน

2Copy Repository URL

  1. คลิกปุ่ม "Code" (สีเขียว)
  2. เลือก "HTTPS"
  3. คัดลอก URL เช่น: https://github.com/your-username/MyProject.git

3เตรียม Local Repository

เปิด Command Prompt / Terminal และรัน:

cd C:\Users\Administrator\Desktop\MyProject

4Initialize Git (ถ้ายังไม่มี)

git init

ตั้งค่าชื่อและอีเมล:

git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

5Add Files

git add .

ตรวจสอบสถานะ:

git status

6Commit Changes

git commit -m "Initial commit: Add MyProject VS Code Learning Hub"

7Add Remote Origin

แทนที่ YOUR-USERNAME ด้วยชื่อ GitHub ของคุณ:

git remote add origin https://github.com/YOUR-USERNAME/MyProject.git

ตรวจสอบ remote:

git remote -v

8Push ไปยัง GitHub

git branch -M main
git push -u origin main
⚠️ สำคัญ: ถ้ามี Popup ให้ขอ Authentication ให้ใช้ Personal Access Token (ไม่ใช่รหัสผ่าน)

🌐 ตั้งค่า GitHub Pages

เพื่อให้เว็บไซต์ของคุณสามารถเข้าถึงได้ออนไลน์:

  1. ไปที่ GitHub Repository ของคุณ
  2. คลิก "Settings" (แท็บด้านบน)
  3. ไปที่ "Pages" ในเมนูด้านซ้าย
  4. ตั้งค่า:
    • Source: เลือก "Deploy from a branch"
    • Branch: เลือก "main" และ folder "/(root)"
    • คลิก "Save"
✅ เสร็จเรียบร้อย! GitHub จะสร้าง URL สำหรับเว็บของคุณ เช่น:
https://your-username.github.io/MyProject/

📝 Tips สำหรับการอัพเดตครั้งต่อไป

เมื่อคุณทำการเปลี่ยนแปลงเพิ่มเติม:

# 1. บันทึกไฟล์ทั้งหมด
git add .

# 2. Commit
git commit -m "Update: Description of changes"

# 3. Push
git push origin main

🔐 Personal Access Token (ถ้าต้อง Authentication)

ถ้า Git ขอ Authentication ให้ใช้ Personal Access Token:

สร้าง Personal Access Token:

  1. ไปที่ GitHub → Settings → Developer settings → Personal access tokens
  2. คลิก "Generate new token"
  3. ตั้งค่า Scope: ✅ repo (เลือก all)
  4. คลิก "Generate token" และคัดลอก
  5. ใช้ Token แทนรหัสผ่านเมื่อ Push

🐛 Troubleshooting

❌ ปัญหา: fatal: not a git repository

git init
git add .
git commit -m "Initial commit"
git remote add origin <your-url>
git push -u origin main

❌ ปัญหา: rejected master -> main

git branch -M main
git push -u origin main

❌ ปัญหา: Authentication failed

  • ใช้ Personal Access Token แทนรหัสผ่าน
  • หรือตั้งค่า SSH Key บน GitHub

❌ ปัญหา: GitHub Pages ไม่ทำงาน

  • ตรวจสอบ Settings → Pages
  • ตรวจสอบว่าเลือก Branch ถูกต้อง
  • รอ 1-2 นาที (บางครั้งต้อง refresh)

✅ ตรวจสอบประวัติ Commit

# ดูประวัติ Commit
git log --oneline

# ดูสาขา
git branch -a

# ดู Remote
git remote -v
